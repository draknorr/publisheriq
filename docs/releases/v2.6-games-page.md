# v2.6 Release Notes - Games Page

**Release Date:** January 16, 2026

---

## Overview

Version 2.6 introduces the completely rebuilt **Games Page**, transforming the basic game list at `/apps` into the most powerful game discovery and analysis tool in PublisherIQ. Following the architectural patterns of the Companies page (v2.5), the Games page features advanced filtering, comparison, export, saved views, and novel computed insight metrics.

### Highlights

- **Unified Games View** - Browse games with type toggle (Games/DLC/Demos)
- **12 Preset Views** - One-click discovery patterns including Momentum, Comeback Stories, Hidden Gems
- **12 Quick Filters** - Stackable filters with AND logic
- **33 Customizable Columns** - Toggle visibility across 9 metric categories
- **6 Computed Insight Metrics** - Momentum Score, Sentiment Delta, Active Player %, Review Rate, Value Score, vs Publisher Avg
- **Compare Mode** - Side-by-side comparison of 2-5 games with best/worst highlighting
- **Export** - CSV/JSON export with configurable columns and filter metadata
- **Saved Views** - Save and load filter configurations (max 10)
- **Sparkline Visualizations** - Inline 7-day CCU trend graphs with batch loading

### Two-Path Query Optimization

The main RPC uses a two-path strategy for performance:

| Path | When Used | Performance | vs Publisher Avg |
|------|-----------|-------------|------------------|
| Fast Path | No vs_publisher filter/sort | ~200ms | NULL |
| Slow Path | vs_publisher filter or sort | ~4s | Computed |

## Performance Optimizations

### Query Optimization

| Optimization | Impact |
|--------------|--------|
| Two-path query | Fast path ~200ms vs slow path ~4s |
| Pre-computed content arrays (`app_filter_data`) | Content filtering: seconds â†’ milliseconds |
| Pre-computed filter counts (MVs) | Filter dropdown: ~5s â†’ <10ms |
| Pre-computed aggregate stats (MV) | Summary stats: timeout â†’ <10ms |

| 3-day growth windows | Works with limited CCU snapshot history |
